# Simple WebSocket Server

This repository contains a very simple WebSocket server script designed for learning and basic testing purposes.

## Overview

The goal of this project is to demonstrate how to:
- Send and receive messages in real time
- Handle connection and disconnection events

The script is intentionally minimal and easy to understand.

## Features

- Basic WebSocket connection handling
- Message receiving and sending
- Lightweight and beginner-friendly
- No complex configuration

## Requirements

- Node.js
- npm
- pm2 (optional but recommended, to run the script in background)
